# Online Shopping System

This is a basic Online Shopping System built using Java with object-oriented programming (OOP) principles.

## Features:
- User login system
- Product catalog
- Shopping cart
- Simple terminal-based navigation

## Technologies Used:
- Java
- OOP (Object-Oriented Programming)

## How to Run:
1. Clone the repository
2. Compile and run the `Main.java` file to start the system.
3. Follow the on-screen instructions to navigate through the system.

## How to Contribute:
Feel free to fork the repository, make changes, and submit pull requests!
